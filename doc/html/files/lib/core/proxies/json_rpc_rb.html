write_now(st)

This file defines all the generic code for JSON communication These connections add more than just the sending and recieving of JSON-RPC commands, They create proxy classes that send and recieve the appropriate messages. These connections should be able to handle the sending and recieving of any Ruby objects. Communication is asyncronous. The only ruby construct that it does not support is blocks. But blocks should not be sent over JSON anyways.
