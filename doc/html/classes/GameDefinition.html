Module GameDefinition
In: lib/core/game_definition.rb

This class defines a game type or expansion pack. For instance, there could be a GameDefinition for the Standard version of Settlers or Cities and Knights, SeaFearers, or even a custom game. The standard game definition is found in this file, but all others should be located in the expansions directory
